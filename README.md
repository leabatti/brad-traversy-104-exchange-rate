# Exchange Rate Calculator — Brad Traversy 50 Projects

Simple currency exchange rate calculator using HTML, CSS, and JavaScript.  
This project replicates the exercise from Brad Traversy’s *50 Projects in 50 Days* course.

## Features
- Select currencies from dropdown menus  
- Fetch real-time exchange rates from an API  
- Automatic calculation of converted amounts  
- Swap currencies with one click  

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  
- Fetch API  

## Project Purpose
Practice API requests, DOM updates, and basic financial calculations.

## How to Use
1. Open `index.html` in your browser.  
2. Select two currencies and enter an amount.  
3. The converted value updates automatically.
